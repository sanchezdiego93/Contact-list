Welcome to the ZingChart HTML5 beta

Overview

Thanks for trying ZingChart HTML5 beta. With this new addition to the ZingChart library, you can now render your charts in both HTML5 and Flash to deliver maximum compatibility to your users. This quick start document will help you build your first HTML5 chart and point you in the right direction for further exploration.

Requirements

No plugins are necessary to develop HTML5 charts, though you'll need Adobe Flash Player 9+ to view charts in the documentation and on elsewhere on our website. And as with all versions, no particular IDE or programming environment is required to develop with ZingChart.

Quick Start

  1. Extract package files into a directory

  2. Create a sample file named hellozing.html and include the following code in the <head> tag

    					<script type="text/javascript" src="../js/jquery-1.3.2.min.js"></script>
					<script type="text/javascript" src="../js/strokeText.js">lt;/script>
					<script type="text/javascript" src="../js/strokeText-ZC.js"></script>
					<script type="text/javascript" src="../js/md5-min.js"></script>

					<!--[if IE]><script type="text/javascript" src="../js/excanvas.compiled.js></script><![endif]-->
					<script type="text/javascript" src="../js/zingchart-html5beta-min.js"></script>

                    <script type="text/javascript">
					$(document).ready(function() {
						zingchart.render({
							id 			: 'zingchart',
							width 		: 800,
							height 		: 400,
							dataurl 	: 'graphfile.txt'
						});
					});
                    </script>

    Note Your paths may be different depending on where you installed the scripts.

  3. In hellozing.html create a <div> holder for the chart where you want the graph to be displayed.

                        <div id="zingchart"></div>

  4. Create the graphfile referenced above called graphfile.txt which will contain JSON for your first chart.

                        {
                      "graphset" : [
                      {
                        "type" : "line",
                        "series" : [
                         {
                            "values" : [5, 10, 15, 5, 10, 5]
                         }]
                      }]
                    }

  5. Load hellozing.html in your browser and you're on your way.
